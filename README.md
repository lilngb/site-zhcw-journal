# site-zhcw-journal

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文档形式存放，不指向任何特定网站或域名。仓库本身仅作为内容存储与版本管理的工具。

## 目录说明

```
/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面依赖的资源文件（样式、脚本、图片等）
├── archive/        # 历史版本或不再更新的页面归档
└── README.md       # 本文件
```

- `pages/`：主目录，包含当前可访问的 HTML 页面。
- `assets/`：存放页面引用的公共资源，按需组织。
- `archive/`：存放已被替代或停止维护的旧版本页面，仅保留参考作用。

## 页面归档说明

- 每个 HTML 页面以独立文件存在，相互之间无强依赖关系。
- 页面内容可能随时间更新，历史版本保留在 `archive/` 目录下。
- 仓库不提供任何形式的自动化部署或预览服务，页面仅供本地浏览或自行托管。

## 维护说明

- 本仓库由维护者不定期更新，不保证内容时效性。
- 欢迎提交 Issue 或 Pull Request 以修正错误或补充内容。
- 请勿添加与页面内容无关的文件或链接。
- 所有提交应保持页面结构清晰、资源引用相对路径正确。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/site-zhcw-journal.git
   ```
2. 直接使用浏览器打开 `pages/` 下的 HTML 文件即可查看。
3. 如需部署至 Web 服务器，将仓库根目录作为站点根目录即可。

## 许可

本仓库内容仅供学习与归档使用，不涉及任何商业用途。具体页面可能包含第三方资源，请遵守相应许可协议。
